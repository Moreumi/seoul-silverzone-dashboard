# 서울시 노인보호구역(실버존) 후보지 제안 대시보드

서울시 노인보호구역의 시설 중심 지정 방식이 실제 노인 보행자 사고 발생지와 어긋날 수 있다는 문제의식에서 출발한 데이터 기반 대시보드 프로젝트입니다.

노인 보행자 사고, 고령자 생활인구, 대중교통 이용량, 도로 환경 데이터를 결합하여 서울시 행정동 및 150m 격자 단위의 신규 실버존 후보지를 제안했습니다.

## 프로젝트 개요

* **프로젝트명**: 서울시 노인보호구역(실버존) 후보지 제안 대시보드
* **목적**: 사고 위험과 노인 생활 환경을 함께 고려해 신규 실버존 후보지를 제안
* **산출물**: 단일 HTML 기반 인터랙티브 대시보드
* **분석 단위**: 서울시 행정동 및 150m 격자 셀
* **방법론**: 합성지표 기반 점수화 및 시간 분할 검증

## 핵심 결과

* 2021-2023년 데이터를 기준으로 후보지를 도출하고, 2024-2025년 사고 데이터로 검증했습니다.
* 제안 후보지는 기존 실버존 대비 미래 노인 보행자 사고를 약 **1.65배 더 많이 포함**했습니다.
* 좁은 도로 비율은 생활인구로 보정한 사고 위험과 가장 강한 관련성을 보였습니다.
* 결과는 사고 예방 효과가 아니라, **신규 보호구역 위치의 적합도**를 검토하기 위한 참고 자료입니다.

## 주요 데이터

* 노인 보행자 사고 데이터: 2021~2025년, 9,564건
* 서울 생활인구 데이터
* 대중교통 이용 인원 및 대중교통 시설 데이터
* 서울시 도로구간 및 도로폭 데이터
* 기존 노인보호구역 공간 데이터
* 행정동–법정동 매핑 데이터

## 나의 역할

* 초기 대시보드 v11의 구조와 점수 엔진 단독 설계·구현
* 사고 건수와 생활인구를 결합한 위험 발생률 지표 설계
* 행정동·법정동 코드 매핑 및 공간 데이터 정합 이슈 검토
* 변수 검증을 위한 상관분석, Random Forest, XGBoost, SHAP 분석 설계·해석
* 시간 분할 검증 설계 및 결과 해석
* 최종 v14→v15 단계에서 분석 엔진 개편

  * min-max 정규화에서 z-score 정규화로 전환
  * 실버존 보정 항목 제거
  * 셀 단위 추천 프리셋 및 사고 심각도 기준 토글 구현

## 대시보드 기능

* 노인 보행자 사고 현황 확인
* 자치구·행정동 단위 위험도 비교
* 사용자 가중치 조정 슬라이더
* 사고 건수 및 EPDO 기준 전환
* 150m 격자 단위 신규 실버존 후보 추천
* 후보 위치의 도로명 및 좌표 확인

## 기술 스택

* Python
* pandas, geopandas
* HTML, CSS, JavaScript
* Leaflet
* 공간 데이터 처리 및 좌표계 변환

## 한계

* 본 프로젝트는 보호구역 설치 이후의 사고 감소 효과를 검증한 연구가 아닙니다.
* 결과는 신규 실버존 지정 후보지를 검토하기 위한 위치 적합도 자료입니다.
* 일부 대중교통 데이터는 행정동 단위로 변환하는 과정에서 생활인구 비례 배분을 적용했습니다.

## 향후 개선 방향

* 보행 신호 시간, 교통량, 보도 폭, 불법주정차 등 추가 도로 환경 변수 반영
* 실제 현장조사 결과와의 비교
* 자치구 담당자 관점의 검토·신청 워크플로우 고도화
